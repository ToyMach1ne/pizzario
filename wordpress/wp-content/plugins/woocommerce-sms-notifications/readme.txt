== Description ==

Send SMS notifications to your [WooCommerce][1] customers when order statuses change, and send store administrators an SMS when you get a new order.

You can also send a quick SMS update to your customer from the order status screen.

The plugin works with Clockwork which means you can send SMS notifications to anyone, anywhere in the world. Once you've installed the plugin you'll need a username and password for a [Clockwork API account][2], you can signup online and text messages cost just 5p.

== Installation ==

1. Upload `woocommerce-sms-notifications` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Clockwork API key in the 'Clockwork Options' page under 'Clockwork SMS'
4. Setup your WooCommerce options on the 'WooCommerce' page under 'Clockwork SMS'

You are now ready for your first SMS notified orders!
